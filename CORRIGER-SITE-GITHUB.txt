  CORRECTION EN 2 MINUTES
══════════════════════════════════════════════════

1. Ouvrez :
   https://github.com/nizarsn18/cafe-socco

2. Cliquez sur le fichier  index.html

3. Cliquez sur le crayon  ✏️  (Edit)

4. Utilisez Ctrl+H (Remplacer) et changez :

   css/style.css     →    style.css
   js/three.min.js   →    three.min.js
   js/config.js      →    config.js
   js/main.js        →    main.js

5. En bas : vert  "Commit changes"

6. Attendez 1 minute, puis sur le site appuyez Ctrl+F5
   https://nizarsn18.github.io/cafe-socco/

Le site sera noir/doré comme sur votre PC !
